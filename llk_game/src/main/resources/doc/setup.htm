:)，人懒了一点，没有做界面，需要手动修改一下配置文件就行了。

文件应该与当前的.jar文件在同一目录下，并且文件名为kyodai.ini(注意大小写，特别是linux平台)，如果文件不存在，自己可以做一个

文件中只有4句话，如下，#表示注释

------------- kyodai.ini -------------

# 控制前景音乐，Music=1是音乐打开，0反之
Music = 0

# 控制间效，目前没有作用:)
Sound = 0

# 控制游戏的难度级别，0最简单，4最复杂，0~4以外的无意义
LevelIndex= 0

# 动画效果，在1~10之前，1最快，10最慢
Animate = 10

------------- kyodai.ini -------------
